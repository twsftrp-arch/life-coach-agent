# Demo Prompts

화면 녹화에서는 아래 두 가지 테마를 순서대로 시연하세요.

## Demo 1

```text
용감한 아기 고양이 이야기
```

확인할 장면:

- Story Writer Agent가 5페이지 동화 JSON을 생성
- Parallel Illustrator Agent가 이미지 1/5부터 5/5까지 병렬 생성
- 최종 출력에 제목, 5페이지 텍스트, 5개 이미지 Artifact가 표시

## Demo 2

```text
달빛 도서관을 여행하는 작은 토끼 이야기
```

확인할 장면:

- 첫 번째와 다른 제목/줄거리 생성
- 5개의 SVG 이미지 Artifact 생성
- Callbacks 진행 상태가 Agent State에 기록
